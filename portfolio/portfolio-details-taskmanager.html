Task Manager – MERN Stack

Project Information

  • Category: Full Stack App
  • Tech Stack: Node.js, Express.js, React.js, MongoDB
  • Year: 2024
  • Source Code: GitHub Repository

Description

A full-featured Task Manager application with separate backend API and React frontend. Supports CRUD operations, filtering, search, and pagination.

Main Features

  • Express.js backend with RESTful routes
  • MongoDB/Mongoose schema design
  • React UI (Component-based)
  • Filtering and pagination
  • Error handling + clean architecture
